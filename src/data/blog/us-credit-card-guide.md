## 前言

我是一個住在台灣、沒有美國身份的投資人。從投資美股開始接觸美國金融市場，陸續開立了國內各券商複委託，以及海外的 Firstrade、Charles Schwab 嘉信、Interactive Brokers 盈透等美股券商帳戶。投資風格偏向價值型與轉機股——像勞斯萊斯（RYCEY）這類深度價值股的長期佈局，台股美股都有涉獵。

因為長年與美國金融體系打交道，我很早就意識到**建立美國信用紀錄**的價值。2024 年 3 月，我辦下了人生第一張美國信用卡——Capital One QuicksilverOne，額度只有 500 美金。14 個月後，我手上已經有 5 張美卡，包括 Chase Sapphire Preferred 和 Amex Marriott Bonvoy Brilliant。

這篇文章記錄我的完整歷程、踩過的坑、以及回頭看會給新手的建議。不是理論教學，是**實戰紀錄**。

## 為什麼台灣人要辦美國信用卡？

你可能會問：台灣的信用卡回饋也不差，何必大費周章辦美卡？

**三個核心理由：**

### 1. 開卡禮太香了

美卡的開卡禮（Sign-Up Bonus）是台灣完全沒有的量級。舉例：
- Amex Marriott Bonvoy Brilliant：**150,000 萬豪點數 + $250 餐飲額度**
- Chase Sapphire Preferred：**100,000 UR 點數**（價值約 $1,250 以上）
- Chase Marriott Bonvoy Boundless：**3 張免費住宿券 + 50,000 點數**

光這三張卡的開卡禮，就足以免費住好幾晚五星級飯店。

### 2. 飯店高階會籍

Amex Brilliant 直接給你**萬豪金卡會籍**，享有：
- 免費早餐
- 房型升等（視空房狀況）
- 延遲退房
- 每年一張 85K 等級的免費住宿券

年費 $650 看起來嚇人，但扣掉 $300 餐飲額度和免費住宿券的價值，實際成本非常低。

### 3. 建立美國信用紀錄

如果你未來有任何可能去美國工作、留學、甚至只是長期旅遊，提前建立信用紀錄會讓很多事情變得容易。租房、辦手機、甚至買車都需要信用分數。

## 申辦前的準備工作

在申請任何美卡之前，你需要先搞定三件事。這是最無聊但最重要的步驟。

### 1. ITIN（Individual Taxpayer Identification Number）

ITIN 是美國國稅局（IRS）發給非美國人的稅務識別號碼，功能類似美國的社會安全碼（SSN）。**沒有 ITIN 就沒辦法申請大部分美卡。**

**申請方式：**
- 準備 W-7 表格 + 護照認證 + 稅務申報（通常用退稅方式）
- 透過 CAA（Certifying Acceptance Agent）辦理最省事
- 費用：大約 2,000 台幣左右（護照認證 + 郵寄 IRS）
- 時間：寄出後約 6-12 週收到 CP565 核准信

> **我的經驗：** 為了申請 ITIN，我花了近 2,000 台幣去申請退稅 6 美金。聽起來很蠢，但這張 CP565 是你整個美卡旅程的入場券。

### 2. 美國手機號碼

推薦 **Ultra Mobile PayGo**，月租只要 $3 美金，可以在台灣收美國簡訊和電話。申請信用卡時需要填美國手機號碼，後續驗證和客服通話也都需要。

### 3. 美國地址

你需要一個可以收信的美國地址。常見選項：
- 美國親友的地址
- 轉運公司地址
- 虛擬信箱服務

> **重點提醒：** Amex 對地址驗證特別嚴格。我在申請 Amex Green 的時候，上傳 CP565 被拒絕，理由是「無法當作地址證明」。Amex 只接受：網路帳單、手機帳單、**信用卡帳單**、財力證明、房屋保險單、水電帳單。最後我是靠 Capital One 的信用卡帳單（上面有美國地址）才過關的。

## 我的申卡歷程：14 個月 5 張卡

以下是我的完整時間線，每張卡的申請過程、踩坑紀錄都包含在內。

### 第 1 張：Capital One QuicksilverOne（2024 年 3 月）

| 項目 | 內容 |
|------|------|
| 申請日期 | 2024/03/17 |
| 核卡日期 | 2024/03/20 |
| 額度 | $500 |
| 開卡禮 | 無 |
| 年費 | $39 |

**為什麼選它：** 當時沒有任何美國信用紀錄，Capital One 是少數願意核卡給零信用 ITIN 申請人的銀行。原本想申請押金卡（Secured Card），結果系統直接給了 QuicksilverOne，跳過了押金步驟。

**申請過程：** 送件後要求補交身份驗證文件（護照、ITIN 信函、W-2 表格），3 天後核卡。

**踩坑紀錄：** 這張卡回饋率只有 1.5%，年費 $39，客觀來說不是好卡。但它的價值不在回饋，而在於**幫你打開美國信用系統的大門**，拿到第一份信用報告和帳單。

### 第 2 張：Amex Green Card — 國際轉卡（2024 年 3-5 月）

| 項目 | 內容 |
|------|------|
| 申請日期 | 2024/03/12 |
| 核卡日期 | 2024/05/02 |
| 額度 | 簽帳卡（無固定額度） |
| 開卡禮 | 40,000 MR 點數 |
| 年費 | $150 |
| 耗時 | 約 7 週 |

**為什麼選它：** 如果你持有台灣的美國運通卡滿一年，可以申請「國際轉卡」（Global Card Transfer），把台灣的持卡紀錄轉到美國。這是**不需要美國信用紀錄就能拿到 Amex 的方法**。

**踩坑紀錄（重要）：** 這張卡花了我整整 7 週才拿到，幾乎都卡在地址驗證。

1. 上傳 CP565（IRS 的 ITIN 核准信）→ **被拒**，Amex 說這不算地址證明
2. 能用的文件：網路帳單、手機帳單、信用卡帳單、財力證明等
3. **解法：** 等 Capital One 第一期帳單寄來（上面有美國地址），上傳後才通過

> **教訓：** 先辦 Capital One 再辦 Amex 國際轉卡，順序很重要。你需要 Capital One 的帳單來當 Amex 的地址證明。

| 項目 | 內容 |

**為什麼選它：** CSP 100K 開卡禮是難得一見的大禮，UR 點數可以轉到多家航空和飯店計畫，靈活度極高。

**申請過程（比 Amex 難很多）：**

1. **4/23：** 透過朋友 referral 連結送件，沒有秒批
2. **4/24：** 打電話追蹤，客服說需要上傳身份證明文件
3. **4/25：** 再次致電，發現 ITIN 在 Experian 的信用報告中末四碼顯示為 **0000**（錯誤）。只有 Equifax 是正確的。經過客服協助更正後才核卡

> **重要發現：** ITIN 在 Experian 的檔案經常出現末四碼 0000 的 bug，這是很多 ITIN 申請人被 Chase 拒絕的原因。Amex 不會有這個問題（他們用不同的驗證方式），但 Chase 風控嚴格很多。

**Chase 風控須知：**
- Chase 審批比 Amex 嚴格非常多
- 核卡後初期建議**小額消費**，不要一下子刷太大筆
- ITIN 使用者要有心理準備打好幾通客服電話

**申請過程（最折騰的一張）：**

第一次被**詐欺拒絕**（Fraud Decline），打了好幾通電話：
- 一般客服線（8830）**無法解除詐欺標記**
- 必須打 **Fraud Specialist 專線（8630）**
- 經過近半小時的溝通，成功移除詐欺標記，但被告知該次申請已為最終決定，需要重新送件

第二次申請同樣觸發詐欺標記，再次打到 8630 處理，最後在 6/7 核卡。

> **踩坑教訓：** 遇到 Chase 詐欺拒絕，不要打一般客服（8830），直接打 **8630 詐欺部門**。帶著你的護照、ITIN、含美國地址的信用卡帳單，耐心解釋就有機會翻案。

## 推薦的申卡策略與順序

根據我的經驗，建議台灣人按以下順序申請：

### Phase 1：建立信用（第 0-6 個月）

```
ITIN 到手
  → Capital One QuicksilverOne（或 Secured Card）
  → 等第一期帳單出來
```

這個階段的目標不是回饋，是**拿到帳單和建立信用報告**。

### Phase 2：Amex 系列（第 3-12 個月）

```
Capital One 帳單當地址證明
  → Amex 國際轉卡（如果有台灣 Amex）
  → 或直接申請 Amex 入門卡
  → 等信用分數到 700+
  → Amex Brilliant 或其他高階卡
```

Amex 對 ITIN 申請人相對友善，審批速度快，適合建立信用的中期。

### Phase 3：Chase 系列（第 12 個月以後）

```
信用紀錄滿 1 年 + 分數 720+
  → Chase Sapphire Preferred（等 100K offer）
  → 30 天後 Chase Boundless 或其他
```

Chase 的 5/24 規則（24 個月內不超過 5 張新卡）和嚴格風控，建議放在最後。

## 常見問題 FAQ

### Q：沒有 SSN 真的可以辦美卡嗎？

可以。ITIN 就是你的替代方案。我全程用 ITIN 辦了 5 張卡。

### Q：ITIN 要花多少錢和時間？

費用約 2,000 台幣（護照認證 + 郵寄），等待時間 6-12 週。建議找 CAA 代辦比較省心。

### Q：信用卡寄到美國地址，我怎麼拿到？

轉運公司或請美國親友代收再寄回台灣。但很多卡（尤其 Amex）可以先綁 Apple Pay / Google Pay 開始消費，不用等實體卡片到手。

### Q：年費那麼高，值得嗎？

要算總價值，不能只看年費數字。Brilliant 年費 $650，但扣掉 $300 餐飲額度和 85K 免費住宿券（價值至少 $300-500），實際成本可能只有 $0-150。第一年還有 150K 開卡禮，絕對是正的。

### Q：Chase 為什麼那麼難？

Chase 風控嚴格、ITIN 在 Experian 容易出錯、又有 5/24 規則限制。建議信用紀錄至少滿 1 年再挑戰。

### Q：一定要照你的順序嗎？

不一定，但強烈建議**先辦 Capital One 再辦 Amex**。因為 Capital One 的帳單是你過 Amex 地址驗證的關鍵文件。
